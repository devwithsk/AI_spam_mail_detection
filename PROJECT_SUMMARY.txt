## ⚡ Quick Start (Choose One)

### Option 1: Automated Setup (Recommended)

**Windows:**
```bash
setup-windows.bat
```

**Linux/Mac:**
```bash
chmod +x setup-unix.sh
./setup-unix.sh
```

### Option 2: Manual Setup

```bash
# Step 1: Train the model
python main.py

# Step 2: Install backend (Terminal 1)
pip install -r requirements.txt
python app.py

# Step 3: Install & run frontend (Terminal 2)
cd frontend
npm install
npm start
```

### Option 3: Docker

```bash
# Using Docker Compose (easiest)
docker-compose up

# Or manual Docker
docker build -t spam-detector .
docker run -p 5000:5000 spam-detector
```


### 2. Health Check
```
GET http://localhost:5000/api/health
Response: {"status": "healthy", "model_loaded": true}
```

### 3. Single Prediction
```
POST http://localhost:5000/api/predict
Body: {"message": "Your email or SMS text"}
Response: {"status": "success", "prediction": "spam/ham", "confidence": 97.5}
```

### 4. Batch Prediction
```
POST http://localhost:5000/api/batch-predict
Body: {"messages": ["msg1", "msg2", "msg3"]}
Response: {"status": "success", "count": 3, "results": [...]}
```

## 🐛 Troubleshooting

### Issue: Model not found
```
Solution: python main.py
```

### Issue: Port already in use
```
Solution: Change port in app.py or package.json
```

### Issue: Cannot connect to API
```
Solution: Make sure both terminals are running
- Terminal 1: python app.py
- Terminal 2: npm start (in frontend directory)
```

### Issue: npm install fails
```
Solution: 
1. Delete node_modules: rm -rf frontend/node_modules
2. Delete lock file: rm frontend/package-lock.json
3. Reinstall: npm install
```
